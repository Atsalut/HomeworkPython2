Планировщик задач (Task Scheduler)

Это консольное приложение позволяет пользователям управлять списком личных задач.
Поддерживаются следующие операции:
1.  Добавление новой задачи: Пользователь указывает название и приоритет (от 1 до 5).
    ID задачи генерируется автоматически, статус по умолчанию "не выполнено".
2.  Просмотр всех задач: Отображается список всех задач с их ID, названием,
    приоритетом и статусом.
3.  Удаление задачи: Задача удаляется из списка по ранее присвоенному ID.
4.  Изменение статуса задачи: Статус задачи (выполнено/не выполнено) может быть
    изменен, указав ID.
5.  Фильтрация задач по статусу: Можно отобразить только выполненные или только
    невыполненные задачи.
6.  Сортировка задач по приоритету: Список задач отображается отсортированным по
    приоритету в порядке убывания (от наиболее высокого к низкому).

Инструкции по запуску:
1.  Убедитесь, что у вас установлен Python (желательно версия 3.9).
2.  Сохраните код приложения в файл с расширением .py (например, oaoa.py).
3.  Откройте терминал или командную строку.
4.  Перейдите в каталог, где сохранен файл.
5.  Запустите приложение, нажав два раза левой кнопкой мыши по файлу, или код, предварительно
    открыв его в компиляторе.
6.  Следуйте инструкциям в консольном меню для управления задачами.

Структура проекта:
- oaoa.py: основной файл с кодом приложения.
  - Класс Task: определяет структуру отдельной задачи.
  - Класс TaskManager: содержит логику управления списком задач.
  - Функция main_menu: отвечает за взаимодействие с пользователем через консольное меню.
